# <a name="query-keywords-c-reference"></a>Parole chiave di query (Riferimenti per C#)

Questa sezione contiene le parole chiave contestuali usate nelle espressioni di query.

## <a name="in-this-section"></a>In questa sezione

|Clausola|Descrizione|
|------------|-----------------|
|[from](from-clause.md)|Specifica un'origine dati e una variabile di intervallo (simile a una variabile di iterazione).|
|[where](where-clause.md)|Filtra gli elementi di origine in base a una o più espressioni booleane separate da operatori logici AND e OR (`&&` o <code>&#124;&#124;</code>).|
|[select](select-clause.md)|Specifica il tipo e la forma che avranno gli elementi nella sequenza restituita quando verrà eseguita la query.|
|[group](group-clause.md)|Raggruppa i risultati delle query in base a un valore di chiave specificato.|
|[into](into.md)|Fornisce un identificatore che può servire come riferimento ai risultati di una clausola join, group o select.|
|[orderby](orderby-clause.md)|Ordina i risultati delle query in ordine crescente o decrescente in base all'operatore di confronto predefinito per il tipo di elemento.|
|[join](join-clause.md)|Unisce due origini dati in base a un confronto di uguaglianza tra due criteri di corrispondenza specificati.|
|[let](let-clause.md)|Introduce una variabile di intervallo per archiviare i risultati delle sottoespressioni in un'espressione di query.|
|[in](in.md)|Parola chiave contestuale in una clausola [join](join-clause.md).|
|[on](on.md)|Parola chiave contestuale in una clausola [join](join-clause.md).|
|[equals](equals.md)|Parola chiave contestuale in una clausola [join](join-clause.md).|
|[by](by.md)|Parola chiave contestuale in una clausola [group](group-clause.md).|
|[ascending](ascending.md)|Parola chiave contestuale in una clausola [orderby](orderby-clause.md).|
|[descending](descending.md)|Parola chiave contestuale in una clausola [orderby](orderby-clause.md).|
