# <a name="null-c-reference"></a>null (Riferimenti per C#)

La parola chiave `null` è un valore letterale che rappresenta un riferimento Null, ovvero un riferimento che non fa riferimento a un oggetto. `null` è il valore predefinito delle variabili di tipo riferimento. I tipi di valore ordinari non possono essere null, ad eccezione dei [tipi di valore nullable.](../builtin-types/nullable-value-types.md)
