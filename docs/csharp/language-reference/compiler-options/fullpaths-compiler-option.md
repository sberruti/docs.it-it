# <a name="-fullpaths-c-compiler-options"></a>-fullpaths (opzioni del compilatore C#)
L'opzione **-fullpaths** fa sì che il compilatore specifichi il percorso completo del file durante l'elenco degli errori e degli avvisi di compilazione.  
  
## <a name="syntax"></a>Sintassi  
  
```console  
-fullpaths  
```  
  
## <a name="remarks"></a>Osservazioni  
 Per impostazione predefinita, per gli errori e gli avvisi generati dalla compilazione viene specificato il nome del file in cui si è verificato un errore. L'opzione **-fullpaths** fa sì che il compilatore specifichi il percorso completo del file.  
  
 Questa opzione del compilatore non è disponibile in Visual Studio e non può essere modificata a livello di codice.
