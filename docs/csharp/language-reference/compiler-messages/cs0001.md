# <a name="compiler-error-cs0001"></a>Errore del compilatore CS0001
Errore interno del compilatore

 Provare a determinare se il compilatore non è riuscito perché non è in grado di analizzare una sintassi imprevista. Se questo errore viene visualizzato ripetutamente, contattare Microsoft.
