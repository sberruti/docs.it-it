# <a name="compiler-error-cs1540"></a><span data-ttu-id="09cfd-102">Errore del compilatore CS1540</span><span class="sxs-lookup"><span data-stu-id="09cfd-102">Compiler Error CS1540</span></span>
<span data-ttu-id="09cfd-103">Impossibile accedere al membro protetto "member" usando un qualificatore di tipo "type1"; il qualificatore deve essere di tipo "type2" (o derivato da esso)</span><span class="sxs-lookup"><span data-stu-id="09cfd-103">Cannot access protected member 'member' via a qualifier of type 'type1'; the qualifier must be of type 'type2' (or derived from it)</span></span>  
  
 <span data-ttu-id="09cfd-104">Una [classe](../keywords/class.md) derivata non è in grado di accedere ai membri protetti della propria classe di base usando un'istanza della classe di base.</span><span class="sxs-lookup"><span data-stu-id="09cfd-104">A derived [class](../keywords/class.md) cannot access protected members of its base class through an instance of the base class.</span></span> <span data-ttu-id="09cfd-105">Un'istanza della classe di base dichiarata nella classe derivata potrebbe, in fase di esecuzione, essere un'istanza di un altro tipo, derivata dalla stessa base ma non correlata in altro modo alla classe derivata.</span><span class="sxs-lookup"><span data-stu-id="09cfd-105">An instance of the base class declared in the derived class might, at run time, be an instance of another type that is derived from the same base but is not otherwise related to the derived class.</span></span> <span data-ttu-id="09cfd-106">Poiché ai membri protetti possono accedere solo i tipi derivati, qualsiasi tentativo di accedere ai membri protetti che potrebbero non essere validi in fase di esecuzione è contrassegnato dal compilatore come non valido.</span><span class="sxs-lookup"><span data-stu-id="09cfd-106">Because protected members can be accessed only by derived types, any attempts to access protected members that might not be valid at run time are marked by the compiler as not valid.</span></span>  
  
 <span data-ttu-id="09cfd-107">Nella classe `Employee` dell'esempio seguente, `emp2` e `emp3` hanno entrambi il tipo `Person` in fase di compilazione, ma `emp2` ha il tipo `Manager` in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="09cfd-107">In the `Employee` class in the following example, `emp2` and `emp3` both have type `Person` at compile time, but `emp2` has type `Manager` at run time.</span></span> <span data-ttu-id="09cfd-108">Poiché `Employee` non deriva da `Manager`, non può accedere ai membri protetti della classe di base, `Person`, usando un'istanza della classe `Manager`.</span><span class="sxs-lookup"><span data-stu-id="09cfd-108">Because `Employee` is not derived from `Manager`, it cannot access the protected members of the base class, `Person`, through an instance of the `Manager` class.</span></span> <span data-ttu-id="09cfd-109">Il compilatore non è in grado di determinare quale sarà il tipo dei due oggetti `Person` in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="09cfd-109">The compiler cannot determine what the run-time type of the two `Person` objects will be.</span></span> <span data-ttu-id="09cfd-110">Di conseguenza, la chiamata da `emp2` e la chiamata da `emp3` causano entrambe l'errore del compilatore CS1540.</span><span class="sxs-lookup"><span data-stu-id="09cfd-110">Therefore, both the call from `emp2` and the call from `emp3` cause compiler error CS1540.</span></span>  
  
```csharp  
using System;  
using System.Text;  
  
namespace CS1540  
{  
    class Program1  
    {  
        static void Main()  
        {  
            Employee.PreparePayroll();  
        }  
    }  
  
    class Person  
    {  
        protected virtual void CalculatePay()
        {  
            Console.WriteLine("CalculatePay in Person class.");  
        }  
    }  
  
    class Manager : Person  
    {  
        protected override void CalculatePay()
        {  
            Console.WriteLine("CalculatePay in Manager class.");
  
        }  
    }  
  
    class Employee : Person  
    {  
        public static void PreparePayroll()  
        {  
            Employee emp1 = new Employee();  
            Person emp2 = new Manager();  
            Person emp3 = new Employee();  
            // The following line calls the method in the Employee base class,  
            // Person.  
            emp1.CalculatePay();
  
            // The following lines cause compiler error CS1540. The compiler
            // cannot determine at compile time what the run-time types of
            // emp2 and emp3 will be.  
            //emp2.CalculatePay();
            //emp3.CalculatePay();  
  
        }  
    }  
}  
```
