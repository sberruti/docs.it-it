# <a name="compiler-error-cs0038"></a>Errore del compilatore CS0038

Impossibile accedere a un membro non statico di tipo outer 'tipo1' tramite il tipo annidato 'tipo2'

 Un campo in una classe non è automaticamente disponibile in una classe annidata. Affinché sia disponibile per una classe annidata, il campo deve essere [statico](../keywords/static.md). In caso contrario, è necessario creare un'istanza della classe outer. Per ulteriori informazioni, vedere [Tipi annidati](../../programming-guide/classes-and-structs/nested-types.md).

 L'esempio seguente genera l'errore CS0038:

```csharp
// CS0038.cs
class OuterClass
{
    public int count;
    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();
            // int count2 = class_inst.count;
            int count2 = count;   // CS0038
        }
    }

    public static void Main()
    {
    }
}
```
