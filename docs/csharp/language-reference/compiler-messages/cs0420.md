# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="a980d-102">Avviso del compilatore (livello 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="a980d-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="a980d-103">'identificatore': un riferimento a un campo volatile non verrà considerato volatile</span><span class="sxs-lookup"><span data-stu-id="a980d-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="a980d-104">In genere, un campo volatile non deve essere passato tramite un parametro `ref` o **out**, perché non verrà considerato volatile all'interno dell'ambito della funzione.</span><span class="sxs-lookup"><span data-stu-id="a980d-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="a980d-105">Esistono eccezioni a questo comportamento, ad esempio quando si chiama un'API con interlock.</span><span class="sxs-lookup"><span data-stu-id="a980d-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="a980d-106">Come per qualsiasi avviso, è possibile usare [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) per disabilitare questo avviso nei rari casi in cui si vuole intenzionalmente usare un campo volatile come parametro di riferimento.</span><span class="sxs-lookup"><span data-stu-id="a980d-106">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="a980d-107">L'esempio seguente genera l'errore CS0420:</span><span class="sxs-lookup"><span data-stu-id="a980d-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```
