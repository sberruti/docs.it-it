# <a name="compiler-warning-level-1-cs1030"></a>Avviso del compilatore (livello 1) CS1030
\#avviso: 'testo'  
  
 Visualizza il testo di un avviso definito con la direttiva [#warning](../language-reference/preprocessor-directives/preprocessor-warning.md) .  
  
 L'esempio seguente mostra come creare un avviso definito dall'utente:  
  
```csharp  
// CS1030.cs  
class Sample  
{  
   static void Main()  
   {  
      #warning Let's give a warning here
   }  
}  
```

La compilazione produce l'output seguente:

```console
example.cs(6,16): warning CS1030: #warning: 'Let's give a warning here'
```
