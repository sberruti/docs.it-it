# <a name="compiler-error-cs0670"></a>Errore del compilatore CS0670
Il campo non può essere di tipo void  
  
 Il tipo di un campo è stato dichiarato come [void](../language-reference/builtin-types/void.md).  
  
 L'esempio seguente genera l'errore CS0670:  
  
```csharp  
// CS0670.cs  
class C  
{  
   void f;   // CS0670  
   // try the following line instead  
   // public int f;  
  
   public static void Main()  
   {  
      C myc = new C();  
      myc.f = 0;  
   }  
}  
```
