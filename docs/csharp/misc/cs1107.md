# <a name="compiler-error-cs1107"></a>Errore del compilatore CS1107
Un parametro può avere un solo modificatore 'modifier name'.  
  
 È un errore per i `this`modificatori di parametro, ad esempio , `ref`, `in`e `out` per apparire più volte in una definizione di parametro.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1107:  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
