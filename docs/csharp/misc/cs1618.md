# <a name="compiler-error-cs1618"></a><span data-ttu-id="d996b-102">Errore del compilatore CS1618</span><span class="sxs-lookup"><span data-stu-id="d996b-102">Compiler Error CS1618</span></span>
<span data-ttu-id="d996b-103">Impossibile creare delegato con 'method'. Dispone di un attributo Conditional</span><span class="sxs-lookup"><span data-stu-id="d996b-103">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="d996b-104">Non è possibile creare un delegato con un metodo condizionale perché il metodo potrebbe non essere disponibile in alcune build.</span><span class="sxs-lookup"><span data-stu-id="d996b-104">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="d996b-105">L'esempio seguente genera l'errore CS1618:</span><span class="sxs-lookup"><span data-stu-id="d996b-105">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
