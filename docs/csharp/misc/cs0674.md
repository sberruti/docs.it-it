<span data-ttu-id="9e121-103">Non usare 'System.ParamArrayAttribute'.</span><span class="sxs-lookup"><span data-stu-id="9e121-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="9e121-104">Usare la parola chiave 'params'.</span><span class="sxs-lookup"><span data-stu-id="9e121-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="9e121-105">Il compilatore C# non consente l'uso di <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; usare [params](../language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="9e121-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="9e121-106">L'esempio seguente genera l'errore CS0674:</span><span class="sxs-lookup"><span data-stu-id="9e121-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
