# <a name="compiler-error-cs0587"></a>Errore del compilatore CS0587

Errore interno del compilatore: fase 'stage'

 Provare a determinare se il compilatore non è riuscito perché non è in grado di analizzare una sintassi imprevista. Se questo non è il caso, provare [a parlare con noi](/visualstudio/ide/feedback-options).

> [!NOTE]
> Questo errore del compilatore non viene più utilizzato in Roslyn.This compiler error is no longer used in Roslyn.
