# <a name="compiler-warning-level-4-cs0422"></a>Avviso del compilatore (livello 4) CS0422

L'opzione /incremental non è più supportata

 La compilazione incrementale (**/incr** o **/incremental**) non è supportata in Cè 2.0.

> [!NOTE]
> Questo avviso del compilatore non viene più utilizzato in Roslyn.This compiler warning is no longer used in Roslyn.
