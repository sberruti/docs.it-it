# <a name="compiler-error-cs0210"></a><span data-ttu-id="6e021-102">Errore del compilatore CS0210</span><span class="sxs-lookup"><span data-stu-id="6e021-102">Compiler Error CS0210</span></span>
<span data-ttu-id="6e021-103">Occorre specificare un inizializzatore nella dichiarazione di un'istruzione fixed o using</span><span class="sxs-lookup"><span data-stu-id="6e021-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="6e021-104">È necessario dichiarare e inizializzare la variabile in un' [istruzione fixed](../language-reference/keywords/fixed-statement.md).</span><span class="sxs-lookup"><span data-stu-id="6e021-104">You must declare and initialize the variable in a [fixed statement](../language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="6e021-105">Per ulteriori informazioni, vedere [Codice unsafe e puntatori](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="6e021-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="6e021-106">L'esempio seguente genera l'errore CS0210:</span><span class="sxs-lookup"><span data-stu-id="6e021-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="6e021-107">L'esempio seguente genera anche l'errore CS0210 perché l' [istruzione using](../language-reference/keywords/using-statement.md) non contiene inizializzatori.</span><span class="sxs-lookup"><span data-stu-id="6e021-107">The following sample also generates CS0210 because the [using statement](../language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test
{  
   static void Main()
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
