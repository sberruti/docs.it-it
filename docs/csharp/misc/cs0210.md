# <a name="compiler-error-cs0210"></a>Errore del compilatore CS0210
Occorre specificare un inizializzatore nella dichiarazione di un'istruzione fixed o using  
  
 È necessario dichiarare e inizializzare la variabile in un' [istruzione fixed](../language-reference/keywords/fixed-statement.md). Per ulteriori informazioni, vedere [Codice unsafe e puntatori](../programming-guide/unsafe-code-pointers/index.md).  
  
 L'esempio seguente genera l'errore CS0210:  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 L'esempio seguente genera anche l'errore CS0210 perché l' [istruzione using](../language-reference/keywords/using-statement.md) non contiene inizializzatori.  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test
{  
   static void Main()
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
