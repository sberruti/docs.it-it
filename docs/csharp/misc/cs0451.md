# <a name="compiler-error-cs0451"></a><span data-ttu-id="6f7c4-102">Errore del compilatore CS0451</span><span class="sxs-lookup"><span data-stu-id="6f7c4-102">Compiler Error CS0451</span></span>
<span data-ttu-id="6f7c4-103">Non è possibile usare il vincolo 'new()' con il vincolo 'struct'</span><span class="sxs-lookup"><span data-stu-id="6f7c4-103">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="6f7c4-104">Quando si specificano i vincoli nel tipo di un oggetto generico, il vincolo `new()` può essere usato solo con vincoli di tipo classe, vincoli di tipo interfaccia, vincoli di tipo riferimento e vincoli del parametro di tipo, ma non con vincoli di tipo valore.</span><span class="sxs-lookup"><span data-stu-id="6f7c4-104">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6f7c4-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="6f7c4-105">Example</span></span>  
 <span data-ttu-id="6f7c4-106">L'esempio seguente genera l'errore CS0451.</span><span class="sxs-lookup"><span data-stu-id="6f7c4-106">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
