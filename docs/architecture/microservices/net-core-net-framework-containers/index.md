# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Scelta di NET Core o NET Framework per i contenitori di Docker

Per la creazione di applicazioni lato server in contenitori di Docker con .NET sono supportati due framework: [.NET Framework e .NET Core](https://dotnet.microsoft.com/download). Queste implementazioni condividono molti componenti della piattaforma .NET e possono condividere codice. Tuttavia, esistono differenze fondamentali tra le due opzioni e la scelta del framework da usare dipende dall'obiettivo che si vuole conseguire. In questa sezione vengono fornite istruzioni su come scegliere il framework.
