 <span data-ttu-id="9defb-109">Una *variabile membro* (dichiarata a livello di modulo, all'esterno di qualsiasi routine) ha in genere la stessa durata dell'elemento in cui è dichiarata.</span><span class="sxs-lookup"><span data-stu-id="9defb-109">A *member variable* (declared at module level, outside any procedure) typically has the same lifetime as the element in which it is declared.</span></span> <span data-ttu-id="9defb-110">Una variabile non condivisa dichiarata in una classe o struttura esiste come copia separata per ogni istanza della classe o della struttura in cui è dichiarata.</span><span class="sxs-lookup"><span data-stu-id="9defb-110">A nonshared variable declared in a class or structure exists as a separate copy for each instance of the class or structure in which it is declared.</span></span> <span data-ttu-id="9defb-111">Ogni variabile di questo tipo ha la stessa durata della relativa istanza.</span><span class="sxs-lookup"><span data-stu-id="9defb-111">Each such variable has the same lifetime as its instance.</span></span> <span data-ttu-id="9defb-112">Tuttavia, una variabile `Shared` dispone di una sola durata, che dura per tutto il tempo in cui l'applicazione è in esecuzione.</span><span class="sxs-lookup"><span data-stu-id="9defb-112">However, a `Shared` variable has only a single lifetime, which lasts for the entire time your application is running.</span></span>
