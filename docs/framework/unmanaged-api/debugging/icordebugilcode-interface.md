# <a name="icordebugilcode-interface"></a>Interfaccia ICorDebugILCode
[Supportato in .NET Framework 4.5.2 e versioni successive]  
  
 Rappresenta un segmento di codice di linguaggio intermedio (IL).  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo GetEHClauses](icordebugilcode-getehclauses-method.md)|Restituisce un puntatore a un elenco di clausole di gestione delle eccezioni (EH) definite per questo IL.|
