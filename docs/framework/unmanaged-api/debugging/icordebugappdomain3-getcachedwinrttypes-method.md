# <a name="icordebugappdomain3getcachedwinrttypes-method"></a>Metodo ICorDebugAppDomain3::GetCachedWinRTTypes
Ottiene un enumeratore per tutti i tipi di Windows Runtime memorizzati nella cache.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetCachedWinRTTypes (   
    [out] ICorDebugGuidToTypeEnum **ppGuidToTypeEnum)  
;  
```  
  
## <a name="parameters"></a>Parametri  
 `ppGuidToTypeEnum`  
 out Puntatore a un oggetto interfaccia [ICorDebugGuidToTypeEnum](icordebugguidtotypeenum-interface.md) in grado di enumerare le rappresentazioni gestite dei tipi Windows Runtime attualmente caricati nel dominio applicazione.  
  
## <a name="requirements"></a>Requisiti di  
 **Piattaforme:** Windows Runtime  
  
 **Intestazione:** CorDebug.idl, CorDebug.h  
  
 **Libreria:** CorGuids.lib  
  
 **Versioni .NET Framework:** [!INCLUDE[net_current_v45plus](../../../../includes/net-current-v45plus-md.md)]
