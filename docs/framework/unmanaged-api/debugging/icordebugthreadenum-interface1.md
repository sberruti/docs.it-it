# <a name="icordebugthreadenum-interface"></a>Interfaccia ICorDebugThreadEnum
Implementa i metodi ICorDebugEnum ed enumera le matrici di ICorDebugThread.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo Next](icordebugthreadenum-next-method.md)|Ottiene il numero specificato di istanze di `ICorDebugThread` dall'enumerazione, a partire dalla posizione corrente.|
