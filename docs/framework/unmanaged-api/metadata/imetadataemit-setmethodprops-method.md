# <a name="imetadataemitsetmethodprops-method"></a>Metodo IMetaDataEmit::SetMethodProps
Imposta o aggiorna la funzionalit√†, archiviata in corrispondenza dell'indirizzo virtuale relativo specificato, di un metodo definito da una precedente chiamata a [IMetaDataEmit::DefineMethod](../../../../docs/framework/unmanaged-api/metadata/imetadataemit-definemethod-method.md).  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT SetMethodProps (
    [in]  mdMethodDef md,
    [in]  DWORD       dwMethodFlags,  
    [in]  ULONG       ulCodeRVA,
    [in]  DWORD       dwImplFlags
);  
```  
  
## <a name="parameters"></a>Parametri  
 `md`  
 [in] Token per il metodo da modificare.  
  
 `dwMethodFlags`  
 [in] Attributi del membro.  
  
 `ulCodeRVA`  
 [in] Indirizzo del codice.  
  
 `dwImplFlags`  
 [in] Flag di implementazione per il metodo.
