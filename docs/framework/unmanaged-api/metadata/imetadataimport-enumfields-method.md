# <a name="imetadataimportenumfields-method"></a>Metodo IMetaDataImport::EnumFields
Enumera i token FieldDef per il tipo a cui fa riferimento il token TypeDef specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT EnumFields (
   [in, out] HCORENUM    *phEnum,
   [in]      mdTypeDef   cl,
   [out]     mdFieldDef  rFields[],
   [in]      ULONG       cMax,
   [out]     ULONG       *pcTokens  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `phEnum`  
 [in, out] Puntatore all'enumeratore.  
  
 `cl`  
 [in] Token TypeDef della classe i cui campi devono essere enumerati.  
  
 `rFields`  
 [fuori] Elenco di token FieldDef.  
  
 `cMax`  
 [in] Dimensione massima della matrice `rFields`.  
  
 `pcTokens`  
 [fuori] Numero effettivo di token FieldDef `rFields`restituiti in .  
  
## <a name="return-value"></a>Valore restituito  
  
|HRESULT|Descrizione|  
|-------------|-----------------|  
|`S_OK`|`EnumFields`restituito con successo.|  
|`S_FALSE`|Non sono presenti campi da enumerare. In tal `pcTokens` caso, Ã¨ zero.|
