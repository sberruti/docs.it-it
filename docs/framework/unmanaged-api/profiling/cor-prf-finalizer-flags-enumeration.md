# <a name="cor_prf_finalizer_flags-enumeration"></a>Enumerazione COR_PRF_FINALIZER_FLAGS
Descrive il finalizzatore per un oggetto.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
typedef enum {  
    COR_PRF_FINALIZER_CRITICAL = 0x1  
} COR_PRF_FINALIZER_FLAGS;  
```
