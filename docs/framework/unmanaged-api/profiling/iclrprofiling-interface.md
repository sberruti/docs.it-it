# <a name="iclrprofiling-interface"></a>Interfaccia ICLRProfiling
Fornisce il metodo [ICLRProfiling:: AttachProfiler](iclrprofiling-attachprofiler-method.md) , che consente a un profiler di connettersi a un processo in esecuzione.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo AttachProfiler](iclrprofiling-attachprofiler-method.md)|Connette il profiler specificato al processo specificato.|
