# <a name="httpwebrequest_httpresponse-field"></a>HttpWebRequest.\_campo HttpResponse

`HttpWebRequest._HttpResponse` è un <xref:System.Net.HttpWebResponse> che contiene i dettagli della risposta HTTP da una richiesta HTTP. Può essere `null` fino a quando non viene ricevuta una risposta HTTP.

## <a name="syntax"></a>Sintassi
  
```csharp  
internal HttpWebResponse _HttpResponse
```

> [!WARNING]
> Il campo `HttpWebRequest._HttpResponse` è interno e non deve essere usato direttamente nel codice.
> 
> Microsoft non supporta l'utilizzo di questo campo in un'applicazione di produzione in qualsiasi circostanza.

## <a name="requirements"></a>Requisiti

**Spazio dei nomi:** <xref:System.Net>

**Assembly:** System (in System. dll)

**Versioni .NET Framework:** Disponibile a partire da 2,0.
