 <span data-ttu-id="81cdb-108">Nel livello intermedio, le entità vengono create dal contesto dati che tiene traccia dello stato e gestisce il caricamento posticipato e l'invio delle modifiche al data-stu-id="81cdb-109">These entities are "attached" to the `DataContext`.</span></span> <span data-ttu-id="81cdb-110">Tuttavia, le entità vengono disconnesse dopo l'invio a un altro livello mediante
