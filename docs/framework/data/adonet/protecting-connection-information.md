```xml  
        userName="MyDomain\UserAccount"
        password="*****" />  
```  
  
 <span data-ttu-id="63534-116">L'account con identità fissa deve avere privilegi limitati che includono solo le autorizzazioni necessarie nel database.</span><span class="sxs-lookup"><span data-stu-id="63534-116">The fixed identity account should be a low-privilege account that has been granted only necessary permissions in the database.</span></span> <span data-ttu-id="63534-117">È inoltre necessario crittografare il file di configurazione in modo da non esporre il nome utente e la password in testo non crittografato.</span><span class="sxs-lookup"><span data-stu-id="63534-117">In addition, you should encrypt the configuration file so that the user name and password are not exposed in clear text.</span></span>
