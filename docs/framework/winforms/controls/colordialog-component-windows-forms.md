# <a name="colordialog-component-windows-forms"></a>Componente ColorDialog (Windows Form)
Il componente <xref:System.Windows.Forms.ColorDialog> Windows Form è una finestra di dialogo preconfigurata che consente all'utente di selezionare un colore da una tavolozza e di aggiungere colori personalizzati a tale tavolozza. È la stessa finestra di dialogo visualizzata in altre applicazioni basate su Windows per selezionare i colori. e costituisce una semplice soluzione, utilizzabile nell'applicazione basata su Windows creata, per evitare di configurare una propria finestra di dialogo.
