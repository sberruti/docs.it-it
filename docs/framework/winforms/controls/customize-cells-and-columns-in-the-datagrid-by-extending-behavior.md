 <span data-ttu-id="ede85-106">La classe <xref:System.Windows.Forms.DataGridViewCell> personalizzata può essere creata derivandola dalla classe di base <xref:System.Windows.Forms.DataGridViewCell> o da una delle relative classi derivate.</span><span class="sxs-lookup"><span data-stu-id="ede85-106">You create a custom <xref:System.Windows.Forms.DataGridViewCell> class by deriving from the <xref:System.Windows.Forms.DataGridViewCell> base class or one of its derived classes.</span></span> <span data-ttu-id="ede85-107">Sebbene sia possibile visualizzare qualsiasi tipo di cella in qualsiasi tipo di colonna, in genere viene creata anche una classe <xref:System.Windows.Forms.DataGridViewColumn> personalizzata per la visualizzazione del tipo di cella.</span><span class="sxs-lookup"><span data-stu-id="ede85-107">Although you can display any type of cell in any type of column, you will typically also create a custom <xref:System.Windows.Forms.DataGridViewColumn> class specialized for displaying your cell type.</span></span> <span data-ttu-id="ede85-108">Le classi di colonna derivano da <xref:System.Windows.Forms.DataGridViewColumn> o da uno dei tipi derivati.</span><span class="sxs-lookup"><span data-stu-id="ede85-108">Column classes derive from <xref:System.Windows.Forms.DataGridViewColumn> or one of its derived types.</span></span>  
  
 <span data-ttu-id="ede85-109">Nell'esempio di codice riportato di seguito viene creata la classe di cella personalizzata `DataGridViewRolloverCell` in grado di rilevare il passaggio in ingresso e in uscita del mouse sui bordi delle celle.</span><span class="sxs-lookup"><span data-stu-id="ede85-109">In the following code example, you will create a custom cell class called `DataGridViewRolloverCell` that detects when the mouse enters and leaves the cell boundaries.</span></span> <span data-ttu-id="ede85-110">Quando il mouse è all'interno dei bordi della cella, viene disegnato un rettangolo interno.</span><span class="sxs-lookup"><span data-stu-id="ede85-110">While the mouse is within the cell's bounds, an inset rectangle is drawn.</span></span> <span data-ttu-id="ede85-111">Poiché questo nuovo tipo è derivato da <xref:System.Windows.Forms.DataGridViewTextBoxCell>, si comporta da tutti gli altri punti di vista come la classe di base.</span><span class="sxs-lookup"><span data-stu-id="ede85-111">This new type derives from <xref:System.Windows.Forms.DataGridViewTextBoxCell> and behaves in all other respects as its base class.</span></span> <span data-ttu-id="ede85-112">La classe di colonna correlata è `DataGridViewRolloverColumn`.</span><span class="sxs-lookup"><span data-stu-id="ede85-112">The companion column class is called `DataGridViewRolloverColumn`.</span></span>  
  
 <span data-ttu-id="ede85-113">Per usare queste classi, creare un form contenente un controllo <xref:System.Windows.Forms.DataGridView>, aggiungere uno o più oggetti `DataGridViewRolloverColumn` all'insieme <xref:System.Windows.Forms.DataGridView.Columns%2A> e popolare il controllo con righe contenenti valori.</span><span class="sxs-lookup"><span data-stu-id="ede85-113">To use these classes, create a form containing a <xref:System.Windows.Forms.DataGridView> control, add one or more `DataGridViewRolloverColumn` objects to the <xref:System.Windows.Forms.DataGridView.Columns%2A> collection, and populate the control with rows containing values.</span></span>
