     <span data-ttu-id="d9d2d-112">Viene visualizzato l' **Editor della raccolta ColumnHeader** .</span><span class="sxs-lookup"><span data-stu-id="d9d2d-112">The **ColumnHeader Collection Editor** appears.</span></span>
