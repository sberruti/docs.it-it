# <a name="staticresource-markup-extension"></a><span data-ttu-id="37592-102">Estensione del markup StaticResource</span><span class="sxs-lookup"><span data-stu-id="37592-102">StaticResource Markup Extension</span></span>
<span data-ttu-id="37592-103">Fornisce un valore per qualsiasi attributo di proprietà [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] cercando un riferimento a una risorsa già definita.</span><span class="sxs-lookup"><span data-stu-id="37592-103">Provides a value for any [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] property attribute by looking up a reference to an already defined resource.</span></span> <span data-ttu-id="37592-104">Il comportamento di ricerca per tale risorsa è analogo alla ricerca in fase di caricamento, che cercherà le risorse che sono state caricate in precedenza dal markup della pagina [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] corrente, nonché altre origini applicazione, e genereranno il valore della risorsa come valore della proprietà negli oggetti Run-Time.</span><span class="sxs-lookup"><span data-stu-id="37592-104">Lookup behavior for that resource is analogous to load-time lookup, which will look for resources that were previously loaded from the markup of the current [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] page as well as other application sources, and will generate that resource value as the property value in the run-time objects.</span></span>
